Project Overview

EGK Company is a crowdsourced package delivery platform developed by Syscom Technologies for the Lebanese market. It solves a real pain point — where traditional courier infrastructure is often unreliable — by connecting senders directly with travelers and delivery agents already heading in the right direction.

Senders post shipment requests with package details, destination, and offered price. Delivery agents browse available trips, accept requests, and earn by carrying the package along their route. Both sides rate each other after delivery — building a trusted, community-driven logistics network across Lebanon and internationally.

How It Works

1
Sender Posts a Shipment Request The sender enters package details, origin, destination, and their offered price — then posts the request to the platform.
2
Delivery Agent Browses & Accepts Delivery agents browse open requests matching their route, review package details and the offered fee, and accept the job that works for them.
3
Package Pickup & Real-Time Tracking The agent picks up the package. The sender tracks the shipment status in real time — from pickup through transit to final delivery.
4
Delivery Confirmation & Rating The agent confirms delivery. The sender receives a notification and confirms receipt. Both sides rate each other — building the platform's trust layer over time.

Technical Challenges

Real-Time Matching Between Senders & Agents

Building a reliable matching system that filters requests by route, package type, timing, and agent rating required careful design of the backend API and real-time notification infrastructure.

Trust & Safety in a P2P Delivery Network

Unlike traditional courier apps, EGK relies on independent travelers — making trust critical. The platform needed identity verification, a meaningful rating system, and admin dispute resolution tools to ensure both sides feel safe.

Cross-Border Shipment Complexity

Supporting international deliveries introduced complexity around customs declarations and cross-border regulations — requiring the app to guide users through legal requirements while keeping the experience simple.
