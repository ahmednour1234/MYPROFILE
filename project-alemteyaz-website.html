Project Overview

This is the public-facing website for Almteyaz Recruitment Company — separate from the internal management system. It serves Saudi employers looking for domestic workers, providing them a clean bilingual experience to browse available candidates, submit their recruitment request, and track it by a unique code — all without creating an account.

The site is built with a mobile-first bilingual approach in blue (#1b3e7a) and gold (#c9a227) brand colors — full Arabic RTL with Tajawal font, English LTR with a single toggle, fixed WhatsApp button throughout, and SEO-optimized for Saudi Arabic search.

🔗 Note: This is the public website. The internal management system (13-stage contract lifecycle, multi-branch, finance, HR) is a separate project — see card 04.

Technical Challenges

Seamless bilingual switching — RTL/LTR without page reload

Every layout direction, font, and text flips between Arabic RTL and English LTR in real time — via data attributes, JS DOM traversal, CSS direction switching, and font swapping with no flicker.

Application tracking without user accounts

Employers track status without registering — unique alphanumeric code per submission, entered on the public tracking page, fed live from the internal management system.

CV library connected to the internal management system

The public CV library is connected to the management system — admin updates (add CV, toggle availability) reflect instantly on the site without duplicating data.
