Project Overview

GFMA London (Global Financial Markets Awards) is an annual prestige awards ceremony celebrating excellence in global finance — hosted at a 5-star luxury venue in Central London. The platform is a complete end-to-end event management system: a public website for attendees and sponsors, an admin dashboard for the organizing team, and a full QR ticketing system covering generation, distribution, and real-time door scanning.

The public website showcases the event, lets sponsors browse and apply for packages, allows voting for nominees, and displays awards and media. The admin dashboard gives the team full control over attendees, packages, voting results, and logistics — all integrated with the QR ticketing system for seamless entry on the night.

Technical Challenges

QR Code Security & Anti-Fraud

A luxury event with high-profile attendees required a tamper-proof ticketing system. Each QR contains a server-side signed UUID validated against the live database — making screenshot copies instantly detectable. The scanner marks each code 'used' on first scan, preventing any second attempt.

Offline Scanner Reliability at the Venue

A live event venue can have unpredictable Wi-Fi. The scanner needed to work reliably with intermittent connectivity — using a local cache of confirmed ticket codes that syncs when connection is restored, ensuring no valid attendee is ever turned away due to connectivity issues.

Luxury Branding Across All Touchpoints

GFMA is a prestigious black-tie event — the platform needed to match that premium positioning at every touchpoint: website, PDF ticket, email, and scanner UI. Delivering a consistent luxury feel across four very different interfaces required careful design coordination throughout the project.
